var vm = new Vue({ 你的参数 })

一个 Vue 实例其实正是一个 MVVM 模式中所描述的 ViewModel，简称vm

响应的数据绑定一般情况下的使用条件

  • 每个 Vue 实例都会代理其 data 对象里所有的属性。注意只有这些被代理的属性是响应的。
  • 如果在实例创建之后添加新的属性到实例上，它不会触发模板更新。
